## 🌌 Overview

Galactic Flow is a feature-rich, space-themed project management web application that helps teams organize missions, track tasks, collaborate in real-time, and analyze productivity with stunning visualizations.

## ✨ Features

### Core Features
- ✅ **Create Projects** - Launch new space missions with deadlines
- ✅ **Task Management** - Add, complete, and delete tasks within projects
- ✅ **Progress Tracking** - Auto-updating progress bars for each project
- ✅ **Drag & Drop** - Reorder tasks like Trello
- ✅ **Local Storage** - All data persists after page refresh
- ✅ **Delete Projects** - Remove completed or cancelled missions

### Advanced Features
- 💬 **Team Chat** - Real-time collaboration with crew members
- 📎 **File Attachments** - Upload and manage project files
- ⏰ **Reminders & Deadlines** - Set and snooze reminders
- 🎤 **Voice Notes** - Simulated voice messages for quick updates
- 📊 **Analytics Dashboard** - Beautiful charts showing task completion and productivity
- 🌗 **Dark/Light Theme** - Toggle between space dark and light themes
- 📦 **Export Data** - Backup all project data as JSON
- 🔔 **Push Notifications** - Browser notifications for deadlines and task updates

### Statistics & Monitoring
- Quick stats widget (projects, tasks, completion rate)
- Live productivity percentage
- Real-time clock in footer

## 🛸 Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Space-themed design with animations, glass-morphism effects
- **JavaScript (ES6+)** - Pure vanilla JS, no frameworks
- **Chart.js** - Interactive analytics visualizations
- **Font Awesome** - Icon library
- **Google Fonts** - Inter font family
- **LocalStorage API** - Client-side data persistence
