## 🌌 Overview

**Galactic Flow** is a feature-rich, space-themed project management web application that helps teams organize projects, track tasks, collaborate in real-time, and visualize progress with beautiful analytics. Built with pure HTML, CSS, and JavaScript - no backend required!

## ✨ Features

### ✅ Core Functionality
- **Create Projects** - Launch new space missions/projects with deadlines
- **Task Management** - Add, complete, and delete tasks within projects
- **Progress Tracking** - Auto-calculated progress bars for each project
- **Local Storage** - All data persists after page refresh
- **Drag & Drop** - Reorder tasks like Trello
- **Delete Projects** - Remove projects when completed

### 💬 Collaboration
- **Team Chat** - Real-time messaging with crew members
- **File Attachments** - Upload files to share with team
- **Voice Notes** - Simulated voice message system
- **Comments** - Add comments to individual tasks

### ⏰ Time Management
- **Deadlines** - Set due dates for projects and tasks
- **Reminders** - Create custom reminders with snooze functionality
- **Notifications** - Browser push notifications for deadlines

### 📊 Analytics
- **Project Charts** - Bar chart showing tasks per project
- **Productivity Dashboard** - Doughnut chart for overall progress
- **Real-time Stats** - Live updates of projects, tasks, and completion rates

### 🎨 UI/UX
- **Space Theme** - Beautiful cosmic design with animations
- **Dark/Light Mode** - Toggle between themes
- **Responsive Design** - Works perfectly on Android, iPhone, tablets, and desktop
- **Smooth Animations** - Fade-in effects, floating elements, and glowing borders

### 🔧 Additional Features
- **Export Data** - Backup all projects and data as JSON
- **Clear Completed Tasks** - Bulk remove finished tasks
- **Quick Stats** - At-a-glance metrics dashboard
- **Live Clock** - Real-time clock in footer
